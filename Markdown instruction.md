## Списки

Чтобы добавить ненумерованные списки, 
необходимо пункты выделить звездочкой (*).

* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки,
необходимо пункты просто пронумеровать.
Например, вот так:
1. Первый пукт
2. Второй пукт
